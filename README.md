# Down-Detector
## A simple script to check whether the website is down or not.

## Requirments:

### packages 

- urllib.request
- sys , argparse
- bcolors

### python > 3.x 

## usage: 

downornot.py  -u < Valid URL which you want to check >

OPTIONS: 

```
-h             --help    
             	 < show the available options >
-u            valid with https:// or http:/
  		< Url which you want to check >
```
